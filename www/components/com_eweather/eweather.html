eWeather COMPONENT

April 2006 :: Version 1.1

A component that displays XML weather feed informations from www.weather.com.
http://www.mambobaer.de/

Features include:

  • eWeather displays weather information from a XML Feed provided by weather.com
  • Users can set their own preferred city (User Profiles)
  • Selectable icon styles
  • Language Support
  • Proxy Support with Proxy Authentication
